### Search For a Package

    deb$ apt-cache search <name>

    rpm$ yum search <name>

### List Repos

	deb: /etc/apt/sources.list

	rpm: /etc/yum.repos.d/*
    # To enable a yum repo set enabled=1

### List All Package Files

    deb$ dpkg -L <package>

    rpm$ rpm -ql <package>

### Find Package That Provides a File

    deb$ dpkg --search <file>

    rpm$ ???

### Update Package List

    deb$ apt-get update

    rpm$ yum check-update

### Upgrade Packages

    deb$ apt-get upgrade

    rpm$ yum update
